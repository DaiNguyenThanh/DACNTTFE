# React App

## Mô tả
Đây là một ứng dụng React được xây dựng để quản lí công việc trong khoa của trường đại học

## Công nghệ sử dụng
- React
- React Router (Quản lý điều hướng)
- Ant Design  (UI Components)
- Axios  (Gọi API)
- Moment.js  (Xử lý thời gian)
- react-beautiful-dnd (UI)

## Cài đặt
Yêu cầu: Node.js >= 18.x và npm .
Cài đặt dependencies: npm install
Chạy ứng dụng: npm run start
Cấu hình môi trường: REACT_APP_API_URL=https://api.example.com

## Build và Deploy
Build ứng dụng cho môi trường production: npm run build
